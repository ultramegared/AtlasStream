# AtlasStream
Modern Streaming Platform built by Atlas Labs
## 🚀 Project Status

AtlasStream is currently in the planning and architecture phase.

## 🎯 Mission

Build a modern, scalable and professional streaming platform with support for:

- Movies
- TV Live
- Documentaries
- User Profiles
- Favorites
- Continue Watching
- Admin Dashboard

## 🛠 Tech Stack

- Backend: NestJS
- Frontend: Next.js
- Mobile: Flutter
- Database: PostgreSQL
- Cache: Redis
- Storage: MinIO
- Containers: Docker
