# Administration API

## Descripción

Este módulo administra toda la plataforma AtlasStream.

Solo usuarios administradores tienen acceso.

—

## Usuarios

### GET /admin/users

Listar usuarios.

—

### DELETE /admin/users/:id

Eliminar usuario.

—

## Contenido

### POST /admin/content

Crear contenido.

—

### PUT /admin/content/:id

Actualizar contenido.

—

### DELETE /admin/content/:id

Eliminar contenido.

—

## Categorías

Administrar categorías.

—

## Géneros

Administrar géneros.

—

## Reportes

- Usuarios activos
- Contenido más visto
- Ingresos
- Suscripciones
- Estadísticas generales

—

## Seguridad

Todos los endpoints requieren permisos de administrador.
