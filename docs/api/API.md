# AtlasStream API

## Descripción

Este documento define la estructura general de la API de AtlasStream.

La API será utilizada por las aplicaciones Web, Android, iPhone y Smart TV.

—

## Módulos

- Authentication
- Users
- Profiles
- Content
- Categories
- Genres
- Reviews
- Favorites
- History
- Continue Watching
- Notifications
- Subscriptions
- Administration

—

## Objetivos

- Centralizar toda la comunicación entre clientes y servidor.
- Mantener una arquitectura REST.
- Facilitar futuras integraciones.
- Garantizar seguridad y escalabilidad.
