# Subscriptions API

## Descripción

Administra los planes y el estado de la suscripción.

—

## Endpoints

### GET /subscriptions

Obtener información de la suscripción actual.

—

### GET /subscriptions/plans

Obtener todos los planes disponibles.

—

### POST /subscriptions

Contratar un plan.

—

### PUT /subscriptions

Cambiar de plan.

—

### DELETE /subscriptions

Cancelar la suscripción.

—

## Información

- Plan actual
- Fecha de renovación
- Método de pago
- Estado
- Historial de pagos
