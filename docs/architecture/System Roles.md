# System Roles

## Descripción

Este documento define los roles oficiales del sistema AtlasStream y los permisos asignados a cada uno.

—

## SUPER_ADMIN

**Descripción:**
Control total de la plataforma.

**Permisos:**
- Administrar todo el sistema.
- Crear y eliminar administradores.
- Gestionar usuarios.
- Gestionar contenido.
- Gestionar suscripciones.
- Configurar seguridad.
- Acceso completo a estadísticas.
- Configuración global.

—

## ADMIN

**Descripción:**
Administrador general de la plataforma.

**Permisos:**
- Gestionar usuarios.
- Gestionar películas.
- Gestionar series.
- Gestionar TV en vivo.
- Gestionar categorías.
- Revisar reportes.

—

## CONTENT_MANAGER

**Descripción:**
Responsable del catálogo multimedia.

**Permisos:**
- Agregar contenido.
- Editar contenido.
- Eliminar contenido.
- Administrar géneros.
- Administrar actores.
- Administrar directores.

—

## MODERATOR

**Descripción:**
Modera la actividad de los usuarios.

**Permisos:**
- Revisar reportes.
- Suspender usuarios.
- Eliminar comentarios.
- Gestionar denuncias.

—

## SUPPORT

**Descripción:**
Atención al cliente.

**Permisos:**
- Ver usuarios.
- Ayudar con suscripciones.
- Gestionar incidencias.
- Restablecer cuentas.

—

## PREMIUM_USER

**Descripción:**
Usuario con suscripción Premium.

**Permisos:**
- Acceso a contenido Premium.
- Continuar viendo.
- Favoritos.
- Múltiples perfiles.
- Descargas (futuro).

—

## USER

**Descripción:**
Usuario estándar.

**Permisos:**
- Crear cuenta.
- Iniciar sesión.
- Ver contenido permitido.
- Administrar su perfil.
- Crear favoritos.
- Historial de reproducción.

—
