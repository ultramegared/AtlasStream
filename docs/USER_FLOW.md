# AtlasStream User Flow

## Objetivo

Definir el recorrido que realiza un usuario desde que abre la aplicación hasta que consume contenido.

—

## Flujo Principal

1. Abrir AtlasStream.

2. Splash Screen.

3. Verificar si existe una sesión activa.

4. Si NO existe:
   - Iniciar sesión.
   - Registrarse.

5. Si SÍ existe:
   - Seleccionar perfil.

6. Pantalla principal (Home).

7. Buscar o explorar contenido.

8. Seleccionar contenido.

9. Ver detalles del contenido.

10. Reproducir.

11. Guardar automáticamente el progreso.

12. Agregar a Favoritos (opcional).

13. Salir o continuar explorando.
