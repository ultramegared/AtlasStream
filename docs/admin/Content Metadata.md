# Content Metadata

## Overview

Metadata describes every piece of information associated with a content item.

—

## Basic Information

- Title
- Original Title
- Description
- Short Description
- Tagline

—

## Classification

- Genre
- Category
- Content Type
- Age Rating

—

## Production

- Director
- Cast
- Studio
- Country
- Release Year
- Duration

—

## Languages

- Original Language
- Available Audio Languages
- Available Subtitle Languages

—

## Media

- Poster
- Banner
- Thumbnail
- Trailer
- Main Video

—

## Publication

- Status
- Visibility
- Publish Date

—

## SEO

- Slug
- Keywords
- Meta Description

—

## Notes

Metadata should be reusable across the entire platform.
