# Content Management

## Overview

The Content Management module is responsible for creating, editing, publishing, organizing and managing all multimedia content available on AtlasStream.

This module allows administrators to manage:

- Movies
- Series
- Seasons
- Episodes
- Live TV Channels
- Cartoons
- Documentaries
- Trailers

—

## Objectives

- Simple and fast administration.
- Multilanguage support.
- Easy publishing workflow.
- Reusable content structure.
- Optimized for future scalability.

—

## Main Features

- Create content.
- Edit content.
- Delete content.
- Publish or unpublish.
- Schedule publication.
- Upload posters.
- Upload banners.
- Upload trailers.
- Upload videos.
- Manage categories.
- Manage genres.
- Manage actors.
- Manage directors.
- Manage countries.
- Manage languages.
