# Content Structure

## Content Types

AtlasStream supports multiple content types.

### Movie

A standalone video.

### Series

A collection of seasons and episodes.

### Live TV

Live streaming television channels.

### Cartoon

Animated content.

### Documentary

Educational or informational content.

### Trailer

Promotional video for other content.

—

## Shared Fields

Every content type shares the following information:

- ID
- Title
- Description
- Poster
- Banner
- Genre
- Language
- Country
- Release Date
- Status
- Visibility
- Created At
- Updated At
